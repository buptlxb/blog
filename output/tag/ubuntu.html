Ubuntu 12.04 Ethernet Controller Problem

 Sun 09 March 2014 by ictlxb

Problem

Ethernet controller can not detect the cable.

Solution

wget http://r8168.googlecode.com/files/r8168-8.037.00.tar.bz2
tar xjf r8168-8.037.00.tar.bz2
cd r8168-8.037.00
./autorun.sh

apt-get error
Sat 01 March 2014 by ictlxb

今天在安装软件时遇到了apt-get发生的一些问题，导致无法安装成功，错误如下：

E:Encountered a section with no Package: header 
E:Problem with MergeList /var/lib/apt/lists/cn.archive.ubuntu.com_ubuntu_dists_natty_main_binary-i386_Packages
...

解决方法如下：

sudo rm /var/lib/apt/lists/* -vf
sudo apt-get update

最后耐心等待即可。
